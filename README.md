# Cloud Chat App

A real-time messaging application developed using Firebase services including Authentication and Firestore Database. The app allows users to communicate instantly with a modern and responsive interface.

## Features
- User Authentication
- Real-time Messaging
- Firebase Firestore Database
- Responsive UI
- Cloud-based Chat System

## Technologies Used
- Firebase
- HTML/CSS/JavaScript (or React if used)
- Firestore Database
- Firebase Authentication
